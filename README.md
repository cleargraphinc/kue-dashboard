# kue-dashboard

## Usage

Available on Docker Hub as [pavlov/kube-dashboard](https://hub.docker.com/r/pavlov/kube-dashboard).

    $ docker run -e REDIS_URL=redis://127.0.0.1:6379 -t pavlov/kue-dashboard

## Development

    $ make build
    $ make push
